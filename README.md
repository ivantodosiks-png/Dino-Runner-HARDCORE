# Dino-Runner-HARDCORE

## Beskrivelse
Dino Runner er et nettleserspill laget med HTML, CSS og JavaScript. Spilleren styrer en dinosaur som hopper over kaktuser og hindringer for å overleve så lenge som mulig. Poeng øker over tid og for hver hindring som passeres.

## Funksjoner
- Fullskjerm Canvas
- Spilles direkte i nettleseren
- Kontroller: Mellomrom eller pil opp for å hoppe
- Hindringer dukker opp tilfeldig
- Poeng og hastighet øker gradvis
